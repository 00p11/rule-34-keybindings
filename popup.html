Key bindings for Rule 34

By 00p11

How does it work?

You can copy any list of post from the site (N) and after that browser at it using L (forwards) and K (backwars).

  • N: create new list

  • A: add items to the existing list

  • L: move forvard in the list

  • K: move backword in the list

  • J: scroll down to the image

  • M: open the image

  • D: download the image

  • B: download the image and put it into your favorites

  • P: play the video

  • Q: console log id array and veiwing location

  • R: deletes stored list

  • H: console log this list

If are you running ito some isues try to check browser console (by right clicking and choosing inspect or by prssing F12), all actions have debug output there.
